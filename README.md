Двусвязный список без использования стандартных коллекций, безопасное управление памятью через систему умных указателей Rc, Weak и RefCell. 
Реализованы основные функции для работы с двусвязным списком: удаление и добавление элемента в конец и начало, по индексу, получение индекса по значению.
Проект включает в себя поддержку стандартных итераторов для обхода в прямом направлении, 
реализацию служебных методов len() и is_empty(), 
а также комплексное модульное тестирование всех операций, включая граничные случаи с пустыми узлами.
